# Multiverse Speedrun Game

Welcome to the Multiverse Speedrun Game project! This repository showcases a dynamic speedrun game built using the Godot Engine, where players can test their skills and compete across various realms and unique challenges.

## Introduction

Imagine a multiverse speedrun game where you can test your skills and compete with others across various realms and unique challenges. In this game, you are not just a player; you are a master of diverse abilities, racing against time and other players to conquer the multiverse. This game is developed using the Godot Engine, ensuring a robust and flexible gaming experience.

## Level 0 Tutorial

The repository contains Level 0, a comprehensive tutorial designed to introduce players to the core mechanics and gameplay elements of the Multiverse Speedrun Game. In this level, you will learn:

- Basic movement and controls
- Fundamental speedrunning techniques
- Introduction to realm-specific challenges

This tutorial serves as the perfect starting point for new players to get acquainted with the game's dynamics before diving into more complex realms.

## Play the Game

The full game is available on [itch.io](https://karma-acg.itch.io/verserunners). Head over to the link to start your adventure in the multiverse!
